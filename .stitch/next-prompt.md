# FoodSharePoint Vendor Dashboard - Premium Revamp

Design a "perfect" management dashboard for a Vendor.

**STITCH DESIGN SYSTEM:**
- Professional marketplace management aesthetic.
- Primary Color: Emerald Green (`#10b981`).
- Accent Color: Warm Terracotta (`#c2410c`).
- Border Radius: `0.75rem` (xl).
- Layout: Sidebar navigation with a spacious main content area.

**Page Sections:**
1. **Stats Overview (Top Cards)**: 4 cards showing: "Total Earnings", "Active SharePoints", "Total Portions Shared", "Avg Rating". Use subtle background tints for each card.
2. **Main Navigation (Sidebar)**: Items like "Overview", "My SharePoints", "Bookings", "Earnings", "Livestreams", "Settings".
3. **Active SharePoints Management**: A list or grid of currently booking events with "Portions Sold" progress bars, "Manage Livestream" buttons, and "Edit" options.
4. **Recent Bookings Table**: A clean table showing customer name, portions booked, total price, and status (Paid, Delivered, etc.).
5. **Upcoming Livestreams Panel**: A section to schedule or start a cook-along session.
6. **Reviews & Feedback Feed**: A sidebar or section showing the latest customer comments.

**DESIGN GOALS:**
- Extreme clarity for management tasks.
- Data-rich but uncluttered.
- Fast access to "Create New SharePoint" and "Go Live" actions.
- Mobile-responsive sidebar (collapsible or bottom nav).
